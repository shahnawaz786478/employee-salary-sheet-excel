# employee salary sheet ( excel project)
this project is a practical payroll management excercise created usking MS-EXCEL
it demonstrates hand-on skills in salary calculation, payroll structure, and basic accounting logic.

## project overview
- designed an employee salary sheet for monthly payroll processing
- automated calculations for earnings and deductions
- structured data for clarity,accuracy, and easy reporting

  ## key features
  - basic salary, HRA, ALLOWANCES, AND deductions
  - net salary auto-calculations using excel formulas
  - clean and professional sheet layout
  - suitable for small office/ practice payroll use

    ## tools and skills used
    - MS EXCEL
    - EXCEL FORMULAS
    - PAYROLL AND ACCOUNTING FUNDAMENTALS

      ## learning outcomes
      - practical understanding of payroll calculation
      - improved excel structuring and formula usage
      - hands-on experience in accounting-related data handling
